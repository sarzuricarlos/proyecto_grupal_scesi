Top 10 Comandos Git Mas Usados

Los comandos que todo desarrollador deberia dominar segun el curso de la SCESI.

#ComandoPara que sirve
1git initInicializa un repositorio Git en la carpeta actual
2git add .Manda todos los archivos al Stage Area
3git commit -mCrea un punto de guardado con los cambios en stage
4git push originSube tus commits al repositorio remoto
5git pull originTrae los cambios del repositorio remoto
6git checkout -bCrea una rama nueva y te mueve a ella
7git merge --no-ffFusiona ramas conservando el historial
8git stashGuarda cambios temporalmente sin hacer commit
9git restoreRevierte un archivo a su estado anterior
10git log --onelineMuestra el historial de commits resumido
